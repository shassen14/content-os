Automated video publishing for content creators

Content OS is a personal desktop application that automates the video publishing workflow — from editing to multi-platform distribution — for a single creator's own accounts.

TikTok Integration

Publishes rendered video files directly to the operator's own TikTok account via the official Content Posting API.

Analytics Dashboard

Pulls performance data (views, likes, shares) from connected accounts into a unified local dashboard.

Self-Hosted

Runs entirely on the operator's local machines. No cloud service, no third-party data storage.

Privacy Policy

Effective date: March 27, 2025

Content OS ("the Application") is a personal, self-hosted desktop tool operated by a single individual for their own content publishing workflow. This policy describes how data is handled.

Data collected

The Application accesses the following data from connected TikTok accounts via the TikTok API, solely for the benefit of the account owner:

  • Basic profile information (username, display name) — used to confirm the correct account is connected
  • Video list and performance statistics (views, likes, comments, shares) — used to populate the local analytics dashboard
  • OAuth access and refresh tokens — stored locally on the operator's own machine

Data storage

All data is stored in a local PostgreSQL database on the operator's private network. No data is transmitted to any third-party server. Tokens and credentials are stored in local configuration files with restricted file permissions.

Data sharing

No data collected through TikTok API access is shared with any third party, sold, or used for advertising. The Application has no user base beyond the single operator.

Data retention

Data is retained locally for as long as the operator chooses to keep it. Revoking TikTok API access in the TikTok app settings will immediately invalidate all stored tokens.

Contact

This is a personal tool with no public user base. For any questions, contact the repository owner via GitHub.
